# 💳 Credit Card Analytics Dashboard (Power BI)

## 📊 Overview :

This project presents an interactive **Power BI dashboard** that analyzes credit card customer behavior and transaction data. It provides key insights into revenue, customer demographics, spending patterns, and financial performance.

The dashboard is divided into two main reports:
- **Customer Report**
- **Transaction Report**

---

## 🎯 Objectives :
- Analyze customer demographics and behavior
- Track revenue, interest, and income trends
- Understand spending patterns across categories
- Identify high-value customer segments
- Enable data-driven decision making

---

## 🧾 Dataset Description :

The project uses two datasets:

### 1. Customer Data (`customer.csv`) :
Contains customer-related information such as:
- Customer demographics (age, gender, marital status)
- Income group
- Education level
- Job type
- Credit card category

### 2. Transaction Data (`credit_card.csv`) :
Includes transaction-related details such as:
- Transaction amount
- Revenue generated
- Interest earned
- Card usage type (Swipe, Chip, Online)
- Expenditure categories

---

## 📌 Dashboard Features :

### 🔹 Customer Report :
- Total Revenue, Interest, and Income KPIs
- Revenue trends over time (weekly analysis)
- Revenue breakdown by:
  - Age group
  - Gender
  - Income group
  - Marital status
  - Education level
  - Job category
- Top-performing states
- Customer segmentation insights

---

### 🔹 Transaction Report :
- Total revenue, interest, transaction amount, and count
- Quarterly revenue and transaction trends
- Revenue distribution by:
  - Card category (Blue, Silver, Gold, Platinum)
  - Transaction type (Swipe, Chip, Online)
  - Expenditure categories (Bills, Entertainment, Fuel, etc.)
- Customer job-based revenue analysis

---

## 🛠️ Tools & Technologies :
- **Power BI** – Data visualization and dashboard creation  
- **CSV Files** – Data source  
- **Data Modeling** – Relationships between datasets  
- **DAX (Data Analysis Expressions)** – Calculations and KPIs  

---

## 📈 Key Insights :
- Blue card category generates the highest revenue
- Swipe transactions dominate usage
- High-income group contributes significantly to revenue
- Businessmen and white-collar customers are top contributors
- Bills and entertainment are the top spending categories

---

## ✅ Conclusion :

This project demonstrates how **Power BI** can be used to transform raw financial data into meaningful business insights. By combining customer demographics with transaction data, the dashboard enables a comprehensive view of credit card usage and revenue generation.

Key takeaways from the analysis include:
- A small number of customer segments (high-income, business professionals) drive a large portion of revenue  
- Transaction behavior (especially swipe usage) significantly impacts overall earnings  
- Spending patterns highlight key categories such as bills and entertainment, which can guide targeted marketing strategies  
- Card category performance shows that entry-level cards (like Blue) contribute the highest revenue volume  

Overall, this dashboard helps stakeholders make **data-driven decisions**, improve customer targeting, and optimize financial strategies.
